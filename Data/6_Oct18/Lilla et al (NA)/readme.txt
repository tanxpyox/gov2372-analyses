Replication Materials for Orr Fowler Huber 2023 AJPS
Is Affective Polarization Driven by Identity, Loyalty, or Substance?


All analysis conducted using R 4.2.1 for macOS
All .R files include install and load commands for required libraries.
The following R packages were used in this analysis:
- dplyr 1.0.9
- estimatr 4.2.0
- ggplot2 3.3.6
- rio 0.5.29
- lme4 1.1-31
- mediation 4.5.0
- reshape2 1.4.4

2. Analysis Files:

The following R scripts reproduce the analysis reported in the main text and SI using the data files listed above.

- analysis_lucid.R (This file contains R code to reproduce the analysis of both original survey experiments as presented in the main text and SI C and D.)

- reanalysis_study2_DiasLelkes2021.R (This file contains R code to reproduce the analysis of study 2 from Dias and Lelkes 2021 as presented in the main text and SI E.)

- reanalysis_study5_DiasLelkes2021.R (This file contains R code to reproduce the analysis of study 5 from Dias and Lelkes 2021 as presented in SI B and H.)

- reanalysis_OrrHuber2020.R (This file contains R code to reproduce the analysis of studies 1, 2, and 3 from Orr and Huber 2020 as presented in SI E)

- analysis_ces.R (This file contains R code to reproduce the analysis of CES data, data_ces.csv, as presented in SI F.)

- analysis_anes.R (This file contains R code to reproduce the analysis of ANES data as presented in SI G.)



3. Codebooks:
